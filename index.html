    The Caesar Cipher is one of history's earliest known encryption techniques, named after Julius Caesar who used it to protect sensitive military correspondence. Each letter in the plaintext is shifted a fixed number of positions along the alphabet.

    Encryption Formula

    Each character is shifted forward by the key k:

    E(x) = (x + k) mod 26

    Decryption Formula

    Reverse the shift to recover plaintext:

    D(x) = (x − k) mod 26

    Key Space

    Only 25 non-trivial keys exist (1–25), making brute-force analysis trivial for modern computers.

    Historical Note

    Suetonius records Caesar using a shift of 3. ROT-13 (shift 13) remains in common use today for obfuscation.
